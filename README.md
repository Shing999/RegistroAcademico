# 📚 Registro Académico - Calculadora de Notas

Aplicación de escritorio desarrollada en **Java Swing** para registrar la información de un estudiante, ingresar sus calificaciones y calcular automáticamente la nota final, indicando si el estudiante **aprueba** o **reprueba** la asignatura.

## 🚀 Características

- Registro de datos del estudiante:
  - Nombre completo
  - Matrícula
  - Carrera

- Registro de calificaciones:
  - Asignaciones
  - Prácticas
  - Parciales
  - Proyecto Final

- Cálculo automático de la nota final.
- Determina si el estudiante:
  - ✅ Aprobó (60 puntos o más).
  - ❌ Reprobó (menos de 60 puntos).
- Validación para impedir caracteres no numéricos en las calificaciones.
- Interfaz gráfica desarrollada con Java Swing.

## ▶️ Ejecución

1. Clonar el repositorio.

```bash
git clone 
```

2. Abrir el proyecto en NetBeans.

3. Ejecutar la clase **Main.java**.

---

## 📖 Funcionamiento

1. Ingresar el nombre del estudiante.
2. Ingresar la matrícula.
3. Escribir la carrera.
4. Introducir las calificaciones.
5. Presionar el botón **Calcular**.
6. El sistema mostrará:
   - Nota final.
   - Estado del estudiante (Aprobado o Reprobado).

---

## ✔️ Validaciones implementadas

- No se permiten letras en los campos de calificaciones.
- Se calcula automáticamente la suma de las calificaciones.
- Se determina el estado del estudiante según la nota obtenida.
